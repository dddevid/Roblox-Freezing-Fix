Roblox Freeze Fix

Esta herramienta soluciona un problema común que causa el congelamiento de Roblox en algunos sistemas Windows ajustando la configuración del cliente para una experiencia de juego más fluida.

Instrucciones de Uso

  1. Descargar el Ejecutable: Descarga RobloxFreezeFix.exe desde la sección de lanzamientos de este repositorio.
  2. Ejecutar el Ejecutable: Haz doble clic en RobloxFreezeFix.exe para ejecutarlo.
  3. Resultado: La herramienta creará o modificará el archivo ClientAppSettings.json en el directorio de configuración del cliente de Roblox, solucionando los problemas de congelamiento durante el juego.
