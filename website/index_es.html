Roblox Freeze Fix

Esta herramienta resuelve un problema común que causa el congelamiento de Roblox en algunos sistemas Windows ajustando la configuración del cliente para una experiencia de juego más fluida.

Instrucciones de uso

  1. Descargar el ejecutable: Descarga RobloxFreezeFix.exe desde la sección de lanzamientos de este repositorio.
  2. Ejecutar el ejecutable: Haz doble clic en RobloxFreezeFix.exe para ejecutarlo.
  3. Resultado: La herramienta creará o modificará el archivo ClientAppSettings.json en el directorio de configuración del cliente de Roblox, resolviendo problemas de congelación durante el juego.
